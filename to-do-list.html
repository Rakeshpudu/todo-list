Block elements

block elements are those which starts from a new line and they take complete width of the screen

examples:

  1. h1-h6 tags
  2. p tags
  3. div tag
  4. section tag

inline elements

inline elements are which starts froms the point where the previous elements ended

examples:

  1. span tag
  2. a tag
  3. strong tag
  4. b tag
